# Har Ghar Pooja - Frontend

React + Vite frontend for the Har Ghar Pooja platform

## Features

- **Responsive Design**: Mobile-first spiritual UI with Saffron Orange theme
- **User Authentication**: Register/Login with JWT
- **Three Dashboards**: User, Pandit, and Admin panels
- **Puja Booking**: Browse and book traditional rituals
- **Real-time Updates**: Dynamic content and status updates

## Tech Stack

- React 18
- Vite
- TailwindCSS
- Axios
- React Router
- Framer Motion
- Lucide React Icons

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Environment

Backend API should be running on `http://localhost:8000`
